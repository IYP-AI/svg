# Contributing to Support Vector Generation (SVG)

Thank you for your interest in contributing! We welcome all contributions including bug reports, feature requests, documentation improvements, and code submissions.

## Getting Started

1. Fork the repository and create your branch from `main`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Set your OpenAI API key as an environment variable (`OPENAI_API_KEY`).
4. Read through our documentation and existing tests in `tests/`.

## Pull Request Process

1. Ensure any new code has tests that cover the changes.
2. Ensure the test suite passes locally (`pytest tests/`).
3. Follow the PEP 8 style guide for Python code.
4. Update the `README.md` and `README_ja.md` with details of changes to the interface, if applicable.
5. Create a Pull Request with a clear description of the problem and your solution.

## Issues

When opening an issue, please use the provided issue templates and include as much detail as possible to help us understand and resolve the problem.
